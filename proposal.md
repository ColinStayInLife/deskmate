# 🤖 DeskMate — 桌面 AI Agent 终端

> 基于 M5Stack CoreS3 SE + DeepSeek + ClawCorp Agent 的物理 AI 助手
> 状态：概念方案 | 日期：2026-04-28

---

## 一、产品定位

### 一句话
**放在桌上的 AI Agent 物理控制台。语音问它、触屏操作、实时看 Agent 状态、不用打开电脑。**

### 解决什么
| 痛点 | 现在 | 有了 DeskMate |
|:----|:-----|:-------------|
| Agent 跑状态要看电脑 | 切窗口看 ClawCorp 日志 | **屏幕看一眼就知道** |
| 审批 Agent 操作 | 必须回电脑操作 | **触屏直接批准/驳回** |
| 查研报/数据 | 打开电脑翻半天 | **语音问一句就行** |
| 通知提醒 | 手机震动可能漏 | **屏幕弹出 + 语音播报** |
| 快速记录想法 | 找手机/开电脑 | **直接对着说话记笔记** |

### F2：语音助手

对着 CoreS3 SE 说话，DeepSeek 理解 + 执行：

| 你问 | 它做 |
|:----|:-----|
| "宁德时代今天怎么样？" | iFinD 拉数据 → 显示行情 + 关键财务 |
| "最近有什么重要公告？" | 查最新公告 → AI 摘要 → 播报 |
| "帮我分析一下光伏板块" | 启动 Agent 任务 → 进度显示到屏幕 |
| "今天有什么待办？" | 查日历/任务列表 → 显示 + 语音 |
| "记一下，宁德目标价..." | 语音转文字 → 存到笔记/飞书 |

### F3：消息通知中心

| 来源 | 触发 | 表现 |
|:----|:-----|:-----|
| ClawCorp Agent | 任务完成 | 屏幕弹出 + 语音："研报已生成" |
| 飞书 | 重要消息 | 显示消息摘要 + 喇叭提醒 |
| iFinD | 股价异动 | 自选股刷红/绿 + 语音播报 |
| 定时任务 | 每日日报完成 | 自动播报：今日投资要点 |

### F4：快捷工具

| 工具 | 操作 |
|:----|:-----|
| 语音速记 | 按住说话 → 自动转文字 → 存 Obsidian |
| 快速搜索 | 语音问 → AI 检索研报库 → 屏幕显示摘要 |
| 倒计时/番茄钟 | 屏幕显示 + 到点提醒 |
| 桌面时钟 | 息屏显示时间/天气/自选股 |

## 四、技术方案

### 4.1 硬件

| 组件 | 规格 | 用途 |
|:----|:-----|:-----|
| **CoreS3 SE** | ESP32-S3, 1.9"触控, 喇叭, 麦克风 | 主机 ✅ 已有 |
| **WiFi** | 2.4GHz | 连接 DeepSeek API + ClawCorp |
| **SD卡** | TF卡槽 | 本地缓存/离线响应 |

**无需额外硬件**。CoreS3 SE 自带的就是全部。

### 4.2 软件栈

```
┌─────────────────────────────────────────┐
│              CoreS3 SE                    │
├─────────────────────────────────────────┤
│  LVGL          ← 屏幕UI框架              │
│  ArduinoAudio  ← 音频采集+播放           │
│  WebSocket     ← 与 ClawCorp 通信        │
│  HTTP Client   ← 调用 DeepSeek API       │
│  JSON          ← 数据解析                │
├─────────────────────────────────────────┤
│            PC/服务器端                     │
├─────────────────────────────────────────┤
│  ClawCorp Agent System                   │
│  DeepSeek API (HTTP)                     │
│  iFinD API                               │
│  Obsidian / Markdown                     │
└─────────────────────────────────────────┘
```

### 4.3 通信协议

```
CoreS3 SE ←→ PC/Server (WebSocket)
  │
  ├─ agent:status       → Agent 状态推送
  ├─ agent:approval     → 待审批操作
  ├─ agent:result       → 任务结果
  ├─ voice:query        → 语音转文字 → DeepSeek
  ├─ voice:response     → TTS 文本
  └─ display:update     → 屏幕内容更新

API (直接调用):
  └─ DeepSeek API HTTP  → 语音理解 / 文字分析
```

### 4.4 DeepSeek 集成方式

```python
# ClawCorp Agent 端代理
# 方式1：CoreS3 录音 → 传 PC → DeepSeek API
# 方式2：CoreS3 → WebSocket → ClawCorp → DeepSeek
# 方式3：CoreS3 直接 HTTP 调 DeepSeek API

# 推荐方式2：ClawCorp 做代理层
# 好处：统一管理 API Key、缓存、日志
```
